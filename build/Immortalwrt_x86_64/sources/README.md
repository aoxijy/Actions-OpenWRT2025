### 🧩 替换 OpenWrt 源码文件的方法说明

📌 **注意：本方法适用于基于源码路径进行文件替换，用于重新编译固件时生效。**

在基于源码编译 OpenWrt 时，有时我们需要替换某些系统文件（如 banner、默认配置等）。可以通过两种方式上传替换文件。

------

## 方法一 ✏️：**手动添加单个文件（按路径替换）**

1. 点击 ➕ **“Add file”** → 选择 ✏️ **“Create new file”**

2. 在路径栏中填写要替换的文件路径和文件名（**必须符合源码中的目录结构**）
    例如：

   ```
   package/base-files/files/etc/banner
   ```

3. 在下方大文本框中粘贴新的文件内容

4. 最下方点击 ✅ **绿色 “Save” 按钮** 保存即可

📝 示例说明：
 若要替换 OpenWrt 登录后的欢迎信息（`banner` 文件），只需使用源码中的路径：

```
package/base-files/files/etc/banner
```

粘贴你自己的 banner 内容保存即可。

------

## 方法二 📤：**批量上传文件夹结构进行替换**

1. 在你电脑上按照源码路径创建对应的文件夹结构，例如替换 `banner` 文件：

   ```
   package/
   └── base-files/
       └── files/
           └── etc/
               └── banner
   ```

   ⚠️ 这里你可以先在 `etc/` 文件夹内创建一个 `banner.txt` 文本文件，编辑好内容后将后缀 `.txt` 删除，变成无扩展名的 `banner` 文件。

2. 回到上传界面，点击 ➕ **“Create new file”** → 选择 📤 **“Upload files”**

3. 将整个 `package` 文件夹直接拖入上传框中

4. 等文件传输完毕后，点击 ✅ **绿色 “Save” 按钮** 完成上传

------

### ✅ 总结对比

| 方式             | 适合场景                       | 操作简要                   |
| ---------------- | ------------------------------ | -------------------------- |
| ✏️ 手动创建单文件 | 替换单个源码文件               | 路径+粘贴内容              |
| 📤 拖入文件夹     | 批量替换多个文件，保持目录结构 | 拖入按源码路径准备的文件夹 |

🧠 **温馨提示**：

- 替换路径必须**严格匹配 OpenWrt 源码目录结构**，否则替换不会生效。
- 修改完毕后，记得重新编译固件才能看到效果。
